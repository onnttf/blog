# 食物营养成分表：读懂它，吃得更明白

买食品，别盯包装正面的"零蔗糖""低脂"。真正靠谱的判断依据只有一个：背面的营养成分表。

## 先看这张表里有什么

### 七项强制标注

新版国标规定，所有预包装食品的营养成分表必须包含以下七项：

| 项目 | 每 100 g | NRV% |
| ---- | -------- | ---- |
| 能量 | 1842 kJ | 22% |
| 蛋白质 | 8.2 g | 14% |
| 脂肪 | 15.6 g | 26% |
| 饱和脂肪 | 5.4 g | 27% |
| 碳水化合物 | 55.0 g | 18% |
| 糖 | 12.0 g | — |
| 钠 | 450 mg | 23% |

表格从左到右三列：项目名称、含量、NRV%。单位上，能量用千焦（kJ），蛋白质/脂肪/碳水/糖用克（g），钠用毫克（mg）。

旧版只强制五项——能量、蛋白质、脂肪、碳水化合物、钠，俗称"1+4"。新版把**饱和脂肪**和**糖**从"可选"升级为"必选"，恰恰是现代人最容易超标的两项。

厂商也可以自愿标注维生素、钙铁锌等额外成分，前提是：强制内容必须更加醒目。

### NRV%：那列百分比怎么用

表头最右侧的百分比叫 NRV%，即营养素参考值百分比（Nutrient Reference Value）。

**NRV** 是国标规定的"普通人一天需要多少"，一组固定数字：

| 营养素 | NRV |
| ------ | --- |
| 能量 | 8400 kJ |
| 蛋白质 | 60 g |
| 脂肪 | 60 g |
| 饱和脂肪 | 20 g |
| 碳水化合物 | 300 g |
| 膳食纤维 | 25 g |
| 钠 | 2000 mg |

**NRV%** 的意思是：吃 100 克这种食品，能满足全天需求的百分之几。

举个例子：某食品蛋白质标注 13.0 g/100 g，NRV% 显示 22%。蛋白质每日参考值是 60 g，13.0 ÷ 60 × 100% ≈ 22%——吃 100 克，就摄入了全天所需蛋白质的五分之一。

NRV% 最大的好处是**不用算**。两款饼干放一起，A 的能量 NRV% 是 27%，B 是 24%，选哪个一目了然。

## 各项营养成分怎么看

### 能量：别光看数字

国内营养成分表强制使用千焦（kJ），不是大家熟悉的"大卡"（千卡，kcal）。换算很简单：

```text
1 kcal = 4.184 kJ
```

千焦除以 4 就是大卡。标 2000 kJ 的食品，热量大约 500 大卡。新版国标允许厂商用"卡""千卡""卡路里"做补充标注，以后可能不用自己算了。

但有个很少有人知道的事：**表上的能量不是测出来的，是算出来的。** 蛋白质 17 kJ/g，碳水 17 kJ/g，脂肪 37 kJ/g，三项相加就是总能量。

**同样标注 1600 kJ 的两包零食，一包热量可能主要来自脂肪，另一包主要来自碳水，吃进身体里完全不是一回事。** 光看能量一栏意义不大，判断食品质量必须结合脂肪和碳水的比例。

### 蛋白质：先看这一栏

大多数人先看热量，再看脂肪。但**蛋白质才是最有参考价值的一栏**。

同等热量下，蛋白质 NRV% 越高，食品的营养密度越好。大致可以这么判断：

- **NRV% ≥ 20%**：高蛋白食品（牛奶、豆制品）
- **NRV% 10%~20%**：中等蛋白来源（全麦面包）
- **NRV% < 10%**：基本是"空热量"

蛋白质还有质量之分。动物蛋白和大豆蛋白属于"完全蛋白"，氨基酸种类齐全；谷物蛋白往往缺一两种。搭配多种来源比单吃一种好。

### 脂肪：注意构成

脂肪是总数值，包含饱和脂肪酸、反式脂肪酸和不饱和脂肪酸。后两者不是强制标注的，但如果生产过程中使用了氢化或部分氢化油脂，国标要求必须标注反式脂肪酸。

注意，国标规定反式脂肪酸 ≤ 0.3 g/100 g 就可以标"0"，所以"0 反式脂肪"不等于没有。新版国标新增了反式脂肪酸每日上限建议：不超过 2.2 克。

### 饱和脂肪：新版新增，重点留意

旧版国标不强制标注饱和脂肪，厂商想标就标。现在不行了。

饱和脂肪的每日参考值是 20 克。饼干、植脂末、人造黄油这类食品，饱和脂肪往往很高。新版国标增加了警示语——"长期过多摄入饱和脂肪可使血胆固醇增高"。

还要注意一个容易混淆的点：即使食品标注了"0 反式脂肪"，饱和脂肪含量往往也很高——这才是更隐蔽的问题。

### 碳水化合物和糖：对照着看

碳水化合物的数值高低本身说明不了太多问题，关键要看它下面紧跟着的"糖"这一栏。两者对照着看：

- **碳水高、糖低** → 主要是淀粉，问题不大（如燕麦）
- **碳水高、糖也高** → 添加糖多，少吃（如甜饼干）
- **碳水低、糖也低** → 低碳食品

典型例子是有些"低脂"酸奶——去掉脂肪后口感寡淡，就加大量糖来弥补。以前碳水数值看着正常，看不出糖有多少。现在糖单独成行，猫腻藏不住了。

新版国标明确了"糖"的定义：果糖、葡萄糖、蔗糖、麦芽糖、乳糖之和。膳食指南建议成人每天添加糖不超过 50 克，最好在 25 克以内。新版国标同样增加了警示语——"长期高糖摄入可增加龋齿和肥胖发生的风险"。

### 钠：不只来自食盐

钠的每日参考值是 2000 毫克，大约相当于 5 克盐。新版国标允许用"盐"字代替"钠"做通俗标注，以后表上可能直接写"盐"，更直观。

要注意，钠不光来自食盐。味精、酱油、加工肉制品、烘焙食品里都藏着它。新版国标也加了警示语——"长期高盐摄入可以导致血压升高"。

## 几个容易踩的坑

### 份量陷阱

营养成分表可以按"每 100 克"标注，也可以按"每份"标注。厂商清楚大多数人不会细算，所以经常在"份"上做文章。

一包薯片净含量 120 g，标注"每份 30 g，能量 670 kJ"——670 kJ 看着不多吧？但整包是四份，吃完 2680 kJ，占全天能量的 32%。

一瓶 500 mL 的饮料，标注"每 100 mL，能量 180 kJ"——看着也不高。喝完一瓶是 900 kJ。

**拿到食品的第一件事：看清表头写的是每 100 g 还是每份，再看整包有多少份。** 这一步不看，后面全白搭。

好消息是，新版国标新增了"份量参考值"附录，对 18 类常见食品规定了推荐份量：坚果 20 g、饼干 30 g、面包 50 g、方便面 100 g、液体饮料 200 mL……同类产品的"份"会越来越统一，以后比较不同品牌会更方便。

### 包装话术，哪些算数

包装正面的"低脂""无糖""高蛋白"受国标约束，有明确门槛：

| 声称 | 条件（每 100 g 或 100 mL） |
| ---- | -------------------------- |
| 无能量 | ≤ 17 kJ |
| 低能量 | ≤ 170 kJ（固体）/ ≤ 80 kJ（液体） |
| 无脂肪 | ≤ 0.5 g |
| 低脂肪 | ≤ 3 g（固体）/ ≤ 1.5 g（液体） |
| 无糖 | ≤ 0.5 g |
| 低糖 | ≤ 5 g |
| 高蛋白 | 蛋白质 NRV% ≥ 20%（固体）/ ≥ 10%（液体） |
| 低钠 | ≤ 120 mg |

这些都是合法的。但打擦边球的营销话术也不少：

- **"零蔗糖"**——不等于无糖。可能加了果糖、麦芽糖浆。不管加什么糖，最终都体现在碳水和"糖"栏里。
- **"非油炸"**——不等于低脂肪。可能通过喷涂油脂达到酥脆口感
- **"全麦""天然""有机"**——跟热量和含糖量没有任何关系

**包装正面是广告，背面才是真相。**

### "0"不是零，"低"也不是绝对低

国标规定了"0"界限值，低于门槛就可以标零：

| 成分 | "0"界限值 |
| ---- | --------- |
| 能量 | ≤ 17 kJ |
| 蛋白质 | ≤ 0.5 g |
| 脂肪 | ≤ 0.5 g |
| 饱和脂肪 | ≤ 0.1 g |
| 反式脂肪酸 | ≤ 0.3 g |
| 碳水化合物/糖 | ≤ 0.5 g |
| 钠 | ≤ 5 mg |

"0 糖"不是真没糖，"0 脂肪"也不是真没脂肪，只是少到可以忽略。

国标还允许标示误差：蛋白质和碳水的实际含量不能低于标示值的 80%，而能量、脂肪、饱和脂肪和钠的实际含量可以高达标示值的 120%。**一包标着"低脂"的食品，实际脂肪可能比标注值还高两成。**

"0"和"低"都是相对的，不是绝对的。

## 实战

比较两款燕麦片：

| 项目 | A 品牌 | B 品牌 |
| ---- | ------ | ------ |
| 能量 | 1650 kJ | 1580 kJ |
| 蛋白质 | 12 g | 10 g |
| 脂肪 | 8 g | 15 g |
| 饱和脂肪 | 2 g | 8 g |
| 碳水 | 62 g | 58 g |
| 糖 | 3 g | 18 g |
| 钠 | 5 mg | 280 mg |

B 的能量和碳水略低，但优势仅此而已。糖 18 g 暴露了它是调味款，饱和脂肪是 A 的四倍。A 的蛋白质更高，脂肪和钠都低得多。结论很明确：选 A。

再看一个常见场景：买牛奶。纯牛奶蛋白质一般在 3.0 g/100 mL 以上（NRV% ≈ 5%）。如果一款"乳饮料"蛋白质只有 1.0 g/100 mL（NRV% ≈ 2%），那它跟牛奶不是一回事。新版国标对液体食品"高蛋白"的要求是 NRV% ≥ 10%，乳饮料根本够不到这条线。

## 记住三件事

养成翻背面看表格的习惯。不需要记公式，不需要算卡路里，做到这三点就够了：

1. **看清单位**——是每 100 g 还是每份，整包有多少份
2. **盯紧 NRV%**——钠和糖越低越好，蛋白质越高越好
3. **别看广告看数据**——包装正面随便印，表里的数字受法律约束

**你不需要成为营养师，只需要学会翻到包装背面。**
